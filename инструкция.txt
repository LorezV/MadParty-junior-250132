Папку alg не стоит рассматривать, они использовалась для написания тестового кода и никак не фигурирует в приложении.

Front и backend запускаются отдельно. Backend никак не зависит от front'да, front зависит от backend'a.

Папка back-node содержит бек-энд часть приложения с алгоритмом.
	- Backend написан на Nodejs, для запуска необходимо установить npm пакеты команой npm install. Запускать команду нужно внутри папки back-node.
	- Сервер запускается на порту 7788.
	- В файлике problems.js лежат распаршенные проблемные темы.

Папка front-end содержит фронт часть приложения. Нужно просто открыть index.html.
	- Запросы отправляются по локальному ip адрессу сервера, поэтому если что-то изменится, то адресс сервера нужно изменить в файлике utils.js
